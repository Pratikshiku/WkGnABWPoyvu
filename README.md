### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（学生）、教师、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 课程信息
5. 公告信息
6. 个人中心


**教师功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 课程信息管理
6. 学生选课管理
7. 学生退课管理
8. 班级信息管理
9. 学生成绩管理
10. 课程评论管理




**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 学生管理
4. 教师管理
5. 课程类型管理
6. 课程信息管理
7. 学生选课管理
8. 学生退课管理
9. 班级信息管理
10. 学生成绩管理
11. 课程评论管理
12. 系统管理（公告信息、系统简介、轮播图管理）
